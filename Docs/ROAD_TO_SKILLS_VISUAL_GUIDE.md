## Accessibility Features

### Keyboard Navigation
- **Tab**: Navigate between cards
- **Enter/Space**: Expand/collapse card
- **Esc**: Close expanded card

### Screen Reader Support
```html
<div
  role="button"
  aria-label="Self-Management skill, currently at Developing level. Click to see tips"
  aria-expanded="false"
  tabindex="0"
>
  [Card content]
```

### Color Contrast
- Text on background: **4.5:1** (WCAG AA)
- Icons paired with text (not color-only)
- Locked state: opacity + lock icon (not just gray)
